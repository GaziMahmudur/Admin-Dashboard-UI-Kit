# 🚀 NexaPanel — Premium Admin Dashboard UI Kit

NexaPanel is a state-of-the-art, developer-friendly Admin Dashboard built for modern SaaS and Enterprise applications. It combines a high-end editorial aesthetic with a powerful React-based logic system.

![NexaPanel Preview](https://via.placeholder.com/1200x600/0f1117/f1f3f9?text=NexaPanel+Admin+Dashboard+Preview)

## ✨ Key Features

- 🌓 **Adaptive Theme System**: Sophisticated Dark and Light modes with `localStorage` persistence.
- 📊 **Interactive Analytics**: Dynamic data visualization using Recharts (Area, Bar, and Pie charts).
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- ⚡ **Performance First**: Zero-dependency build (besides React/Tailwind/Recharts) running entirely via CDN for instant preview.
- 🗂️ **Advanced Data Table**: Integrated real-time search, multi-column sorting, and pagination logic.
- 🎨 **Premium Aesthetics**: Built with the **Plus Jakarta Sans** font and a custom-curated color palette.

## 🛠️ Tech Stack

- **Core**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Typography**: Plus Jakarta Sans (Google Fonts)
- **Icons**: Heroicons (SVG)
